Grid

The Problem

You've got tons of content, each needing different sized vertical columns, and don't know how to quick and easily get it all done.

The Solution

The awesome grid! The grid is built around three key elements: containers, rows, and columns.

  1. Containers create base padding for the page;
  2. rows create a max-width and contain the columns;
  3. and columns create the final structure.

Everything on your page that you don't give a specific structural style to should be within a container, row and column.

How to Use

Using this framework is easy. Here's how your code will look when you use a series of div tags to create vertical columns.

Nesting Support

In the Grid you can nest columns down as far as you'd like. Just embed rows inside columns and go from there. Each embedded row can contain up to number $total-columns defined in _settings.scss.

HTML 

<div class="row">
  <div class="small-2 large-4 columns">...</div>
  <div class="small-4 large-4 columns">...</div>
  <div class="small-6 large-4 columns">...</div>
</div>
<div class="row">
  <div class="large-6 columns">...</div>
  <div class="large-6 columns">...</div>
</div>

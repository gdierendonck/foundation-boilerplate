    • Font icons

    These examples use Foundation icons 3, but feel free to try it with fontawesome, fontello or your own svg icons !

    Icons

    You can customize your icons to be any size, color, style in CSS.

    HTML 

    <i class="fi-[icon] small"></i>
<i class="fi-[icon] medium"></i>
<i class="fi-[icon] large"></i>

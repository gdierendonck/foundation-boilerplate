    • Select

    Style an HTML select element with CSS no JavaScript needed

    A note on using CSS to style select

    Note: From an original idea of Todd Parker - Filament Group Inc.

    How it works

    This styles a native select consistently cross-platform with only minimal CSS. The native select is then styled so it is essentially invisible (no appearance, border, bg) leaving only the select's text visible. There is a wrapper around the select that has the majority of the button styles (gradient, shadow, border, etc.). We then add the custom arrow via a pseudo element to the right.

    How to Use

    This solution is perfect when the button needs to submit a form of some kind.

    Editing the CSS

    SCSS variables for buttons can be found in _select.scss file.

    HTML 

    <label class="wrapper">
  This label wraps the select
  <div class="custom-select">
    <select>
      <option> Apples </option>
      <option> Bananas </option>
      <option> Grapes </option>
      <option> Oranges </option>
    </select>
  </div>
</label>
